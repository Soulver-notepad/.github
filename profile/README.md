# Soulver — Portable Notepad Calculator with Word Calculations

## About Soulver

Soulver provides a complete calculation environment for users who want to compute with words instead of buttons. Soulver combines a notepad interface, natural language calculations, variables, units conversion, and date math. Soulver enables users to write "20 hours at $35/hour" and see "$700". Soulver supports percentages, currency conversion, units (meters, feet, kg, lbs), and variables. Soulver runs on desktop platforms. Soulver is trusted by writers, designers, freelancers, students, and anyone who hates traditional calculators.

## Deep Dive — Calculation Notes

Natural language: "10% of 50", "20 hours at $35/hour", "5 meters in feet"
Variables: assign names to numbers, reuse in multiple lines
Units: length, area, volume, weight, time, speed, temperature, currency
Dates: "days until Christmas", "April 15 + 30 days"
Live results: calculations update as you type, line by line

## Quick Reference for Soulver

| Operation | Soulver Action |
|-----------|----------------|
| New sheet | Cmd+N |
| Calculate selected lines | Cmd+Enter |
| Add variable | Type name = value |
| Convert units | Type "10 meters in feet" |
| Date calculation | Type "today + 30 days" |
| Show result as percentage | Cmd+Shift+P |
| Toggle currency conversion | Cmd+Shift+C |
| Comment line (no calculation) | Type // at start |
| Copy result only | Cmd+Shift+C |
| Open variables panel | Cmd+V |
